Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It defines the structure and layout of a web document by using a variety of tags and attributes.

Each HTML tag servesg a specific purpose, such as defining headings, paragraphs, links, images, forms, and more.<p>For example, the <h1> to <h6> tags define headings of different levels of importance, while the tag defines paragraphs of text

HTML documents are structuredas a hierarchy of elements nested within each other. The <html> tag represents the root of the document, containing two main sections: the<head> section for meta-information and the <body> section for the visible content of the page.

Here basic Example of an Html Document

		 	
		 		
		 		<!DOCTYPE html>
		 		<html>
		 		<head>
		 			<meta charset="UTF-8">
		 			<title>My First HTMl Page;</title>
		 		</head>
		 		<body>
		 			<h1>Welcome TO my Website;</h1>
		 			<p>This is ParaGraph of Text;</p>
		 			<a href="https://www.example.com">Visit Example.com</a>
		 		</body>
		 		</html>

HTML is a foundational technology for web development and is complemented by CSS (Cascading Style Sheets) for styling and JavaScript for dynamic behavior.
