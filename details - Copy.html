More about HTML5

HTML5 is the latest version of the Hypertext Markup Language, the code that describes web pages. It is the fifth and current version of the HTML standard.

  • Improved support for embedding multimedia
  • New form controls
  • Better support for web applications
  • Enhanced semantic elements
More about CSS3

CSS3 is the latest evolution of the Cascading Style Sheets language, bringing new features like rounded corners, shadows, gradients, transitions, and animations, as well as new layouts like multi-columns, flexible box or grid layouts.

More about JavaScript

JavaScript is a versatile programming language that conforms to the ECMAScript specification. JavaScript is high-level, often just-in-time compiled, and multi-paradigm.

JavaScript has dynamic typing, prototype-based object-orientation, and first-class functions.
